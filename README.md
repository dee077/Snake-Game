# Snake Game
object oriented implementation of terminal based snake game

<img src="https://github.com/dee077/Snake-Game-C++/blob/master/gif/snake%20game%20v1.gif">


## How To Play

###  Linux
#### clone the master branch
```
$ git clone https://github.com/dee077/Snake-Game-C++/tree/master
```
#### compile and run
```
$ g++ snake.cpp -o snake-game
$ ./snake-game
```

###  Windows
#### clone the snake_game_windows branch
```
> git clone https://github.com/dee077/Snake-Game-C++/tree/snake_game_windows
```
#### compile and run
```
> g++ snake.cpp -o snake-game
> snake-game
```
